# Hotel Booking System

SE 4458 - Software Architecture & Design of Modern Large Scale Systems Final Project

## Project Description

This project is a microservice-based hotel booking platform similar to Hotels.com.

The system supports:
- hotel search
- hotel booking
- hotel comments and ratings
- notification services
- AI-assisted booking
- hotel administration

## Architecture

Frontend → API Gateway → Microservices → Databases / Cache / Queue
